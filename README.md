Smart Student Management System

Smart Student Management System is a Java-based desktop application developed using Java Swing and MySQL. This system is designed to manage student records efficiently by automating attendance tracking, marks management, attendance percentage calculation, and grade generation. It reduces manual errors and saves time for administrators by providing a centralized and organized platform for academic record management.

Features

Add, Update, Delete Student Records

Attendance Management

Automatic Attendance Percentage Calculation

Marks Entry and Automatic Grade Generation

User-Friendly GUI

Technologies Used

Java (Swing)

MySQL

JDBC

Git & GitHub
